# 🌐 Mobirise Website

This is a responsive website created using Mobirise Website Builder.  
It is a static frontend project built with modern web technologies and optimized for all devices.


---

## 📌 About the Project

This project is a simple and responsive website designed using Mobirise's drag-and-drop interface.

It demonstrates:

- Clean and modern UI design  
- Fully responsive layout  
- Fast-loading static pages  
- Easy customization and deployment  

---

## 🚀 Features

- 📱 Mobile-friendly (responsive design)  
- ⚡ Fast performance (static HTML/CSS)  
- 🎨 Pre-designed UI blocks  
- 🔧 Easy to edit and customize  
- 🌍 Hosted using GitHub Pages  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- Bootstrap (via Mobirise)  
- JavaScript  

---

mobirise-website/
